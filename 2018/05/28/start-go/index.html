시작 Go

On your mark. Get set. GO!

28 May 2018, 23:00
go
syntax-go / fun-go / eg-go

Go 토픽은 제가 Go 언어를 공부하면서 정리한 내용들입니다.

그간 주로 C/C++을 사용해 왔던지라 ‘그에 비해 Go는 이렇더라’라는 내용이 주를 이룹니다. 따라서 내용이 체계적이지 않습니다. 즉, Go 언어를 학습하기 위한 용도로는 적절치 않을 수 있습니다.

syntax-go 태그는 Go 언어의 문법에 대한 내용을 주로 The Go Programming Language Specification을 참고해 정리할 예정입니다.

fun-go 태그는 Go 언어를 공부하면서 시도해 봤던 헛짓거리들에 대한 내용입니다. 간혹 건질만한 것이 있었으면 하는 바람입니다.

eg-go 태그는 개발하면서 자주 만나게 되는 문제들에 대해 Go로는 어떻게 구현하는 것이 좋을지에 대한 코드 조각 (code snippet)들을 모아볼 생각입니다.


모두 열Go 하시길…
